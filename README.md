🔐 CODSOFT Task 3 - Advanced Password Generator

📌 Project Overview

This project is an Advanced Password Generator built using Python and CustomTkinter as part of the CODSOFT Python Programming Internship.

The application generates secure, random passwords based on user-selected preferences. It includes password strength analysis, clipboard support, customizable password length, multiple character options, and password history for improved usability.

---

✨ Features

- 🔐 Generate secure random passwords
- 📏 Adjustable password length (4–32 characters)
- 🔠 Include uppercase letters
- 🔡 Include lowercase letters
- 🔢 Include numbers
- 🔣 Include special symbols
- 💪 Real-time password strength checker
- 📋 One-click copy to clipboard
- 📝 Password generation history
- 🌙 Modern dark-themed interface using CustomTkinter

---

🛠️ Technologies Used

- Python 3
- CustomTkinter
- Tkinter MessageBox
- Random Module
- String Module
